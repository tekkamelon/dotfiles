# Gemini CLI Bridge スキル

このスキルは、他のコーディングエージェント（Claude Code, GitHub Copilot CLIなど）から `gemini-cli` をコマンドライン経由で呼び出し、その強力な解析能力や日本語処理能力を活用するためのものです。

## 用途

- **高度な情報収集**: 大規模なコードベースの検索や依存関係の解析。
- **日本語処理**: コメントの日本語化、日本語による仕様解説、ドキュメント作成。
- **マルチステップタスク**: `gemini-cli` の自律的な実行能力をサブプロセスとして利用。

## 使用方法

基本的な非対話モードの呼び出し方は以下の通りです：

```bash
gemini --prompt "プロンプト内容" --approval-mode plan
```

- `--prompt` (または `-p`): 実行したい指示を渡します。
- `--approval-mode plan`: 読み取り専用モードで実行し、ファイル変更を防止します（情報収集に最適）。

## メリット

1. **日本語の精度**: 日本語のニュアンスを正確に理解し、自然な日本語で回答を生成します。
2. **コンテキスト効率**: `gemini-cli` 自身がコードベースを探索するため、親エージェントが大量のファイルを読み込む必要がありません。
3. **ツールの活用**: `grep_search` や `glob` などの強力な内蔵ツールを間接的に利用できます。
