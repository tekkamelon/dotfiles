# POSIX Shell スキル README

## 概要
POSIX準拠のシェルスクリプトを書くためのスキルです。
ポータブルなsh/bashスクリプト作成、移植性確保、非POSIX構文回避、エラーハンドリング、テスト設計、デバッグをカバー。

キーワード「シェルスクリプト」「sh」「bash」「POSIX」「trap」「getopts」などで使用。
